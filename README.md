# Insurance Charges Linear Regression

This project is a small notebook-based assignment using the Insurance Charges dataset.

The main idea is simple: use Linear Regression to predict insurance charges from features like age, BMI, number of children, smoker status, sex, and region.

The notebook includes:

- a short data exploration
- preprocessing
- a custom Linear Regression implementation with gradient descent
- a comparison with `scikit-learn`
- a short discussion and conclusion

## Files

- `insurance_linear_regression_notebook.ipynb`: the main notebook
- `insurance.csv`: the dataset
- `requirements.txt`: the Python libraries you need

## Install

If you want to run the notebook locally, install the dependencies first:

```bash
pip install -r requirements.txt
```

If `pip` does not work on your system, try:

```bash
python3 -m pip install -r requirements.txt
```

## Run

Open the notebook with Jupyter:

```bash
jupyter notebook
```

Then open `insurance_linear_regression_notebook.ipynb`.

Have fun
